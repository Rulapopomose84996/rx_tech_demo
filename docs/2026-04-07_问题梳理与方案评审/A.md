# A表 一次性装订表文档

## 1. 适用范围

A表用于定义前端设备一次性下装的慢变资源与模板参数，主要包括：

1. 三脉冲波形模板参数。
2. 接收与采样模板参数。
3. 工作模式默认配置。
4. 波形资源与滤波资源的存储映射。

---

## 2. 设计原则

### 2.1 总体原则

1. A表采用“表头 + 分区参数块”的固定结构，便于 FPGA RAM 预分配与硬件状态机解析。
2. 多脉冲相关字段统一采用数组化结构设计，索引体系统一冻结为 `PulseIndex = 0..2`。
3. 全表采用小端编码，并按 4B 自然对齐组织。
4. 物理量量化口径与 [前端感知通信协议V1.0](docs/2026-04-07_问题梳理与方案评审/前端感知通信协议V1.0.md) 保持一致，避免上位机、前端固件和表设计之间出现单位歧义。
5. A表对三脉冲采用“模板属性镜像 + 存储资源映射”双轨表达：量化字段用于配置、核对与回显，精细波形实现以存储资源映射为最终执行依据。

### 2.2 与其他数据表的边界

1. A表负责一次性装订的波形资源、采样模板和工作模式默认值。
2. D表负责波位级调用、角度指向、周期组合、实时 MGC 与采样执行参数。
3. C表负责状态回传与波位数据回传，不回写 A 表内容。
4. A表中的 `脉冲手动增益代码[3]`、`脉冲采样点数[3]`、`采样起始点`、`采样长度点数` 必须与 C 表、D 表采用同一索引规则和语义口径。

---

## 3. 总体结构

A表采用如下固定结构：

```text
A表 一次性装订表
├─ 表头（固定 16B）
├─ 三脉冲波形模板区（固定 32B）
├─ 接收与采样模板区（固定 16B）
├─ 工作模式与资源选择区（固定 16B）
└─ 存储映射区（固定 32B）
```

长度关系如下：

**A表总长度 = 16B + 32B + 16B + 16B + 32B = 112B**

### 3.1 多脉冲索引规则

当前版本统一冻结如下：

1. `PulseIndex = 0..2`
2. `0` 表示第一脉冲
3. `1` 表示第二脉冲
4. `2` 表示第三脉冲

所有带 `[3]` 的字段均遵循上述索引规则，A表、C表、D表不得各自定义不同的脉冲编号体系。

### 5.3 本区补充说明

1. `脉冲发射脉宽代码[3]` 与 [时序说明](docs/2026-04-07_问题梳理与方案评审/时序说明.md) 中的纳秒级边沿冻结值属于不同层面的表达：前者用于模板量化，后者用于执行时序冻结。
2. 当量化字段不足以完整表达精细波形时，应以“脉冲体制代码[3] + 存储映射区”共同确定最终波形资源。
3. `脉冲采样点数[3]` 仅定义单脉冲模板默认采样规模，不等同于 C 表分包上传时某个物理包中的 IQ 数据字节长度。

3. 时序说明中“AD 采样波门”给出的冻结时刻属于物理执行时序，A表中的本区字段用于模板配置和软件可见性，不替代硬件内部补偿量。 |

### 8.3 本区补充说明

1. 三脉冲存储字段全部采用数组化表达，避免重复定义“第一脉冲/第二脉冲/第三脉冲”三套地址和长度字段。
2. `脉冲波形存储基地址[3]` 与 `脉冲波形字节长度[3]` 描述的是资源文件或波形块的存储布局，不直接等同于波门时序。
3. `A表业务校验码` 用于业务体自校验；若外层还使用 B 表整帧 CRC，则两者分别承担“业务内容一致性”和“传输帧完整性”两层校验职责。

### 9.2 统一约束

1. `PulseIndex = 0..2` 在 A表、C表、D表中统一采用 0-based，不允许出现 1-based 变体。
2. `脉冲手动增益代码[3]` 的实际换算公式为：**实际增益 = 编码值 × 0.5 dB**。
3. `脉冲工作频点代码[3]` 的实际换算公式为：**实际频率 = 编码值 × 10 MHz**。
4. `采样波门起始时延代码`、`采样波门持续时长代码` 的实际换算公式为：**实际时延 = 编码值 × 0.01 μs**。
5. 当“模板量化字段”与“资源波形文件”存在表达粒度差异时，应以存储资源映射所指向的实际资源为准，A表量化字段用于配置核对与接口一致性。

## 10. 冻结结论

本版 A 表冻结如下：

1. A表采用固定 **112B** 结构。
2. 全文统一使用“偏移、字段名称、数据类型、字长、取值范围、量化口径、注释说明”七列字段表。
3. 多脉冲字段统一采用数组化表达，索引冻结为 `PulseIndex = 0..2`。
4. 存储映射区中的三脉冲地址、长度和存储区号全部采用数组化结构。
5. MGC、频点、带宽、脉宽、周期、时延等量化口径与通信协议 V1.0 保持一致。
6. A表中的采样点类字段与 C表、D表保持统一语义。
7. A表只定义慢变资源和模板，不承载波位级实时调度信息。
